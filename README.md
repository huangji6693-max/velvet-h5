# Velvet H5 Demo

> Touch what was touched. 私藏，流转。

这是 Velvet 的 H5 编辑式奢华 demo，自动从主仓库 `velvet/h5-demo` 同步部署到 GitHub Pages。

## 在线访问

https://huangji6693-max.github.io/velvet-h5/

## 自定义 API

trycloudflare URL 不稳定时可以通过 query string 临时切换：

```
https://huangji6693-max.github.io/velvet-h5/?api=https://your-new-tunnel.trycloudflare.com
```

切换后会写入 localStorage 永久生效。
